Questions:

1. List out the different ways to select an item.

  • Element selectors (specific HTML tags).
  • Class selectors (using a class attribute).
  • ID selectors (selecting by its ID)
  • Combinator selectors (relationships between elements) include child, sibling, and descendant selectors.

2. What is the difference between an ID and a Class?

  • An ID is a unique identifier that can only be applied to one element on a page.
  • A Class is also a unique identifier but can be applied to multiple elements.

3. What is the difference between a SPAN tag and a DIV tag?

  • A SPAN tag is an inline element used to style specific parts of text within a line.
  • A DIV tag is a block-level element used to create larger sections or divisions on a webpage.

4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

A DIV tag is a container element used to group content for styling purposes. But HEADER, MAIN, or FOOTER are HTML tags that specifically telling the content of a webpage

5. What are the different ways to select a color?

  • Color names (keywords).
  • Hexadecimal values.
  • RGB (Red, Green, Blue) values.
  • RGBA (RGB with alpha transparency).
  • HSL (Hue, Saturation, Lightness) values.
  • HSLA (HSL with alpha transparency).
