Questions

1. What is the difference between block tags and inline tags?

- Block tags create distinct, new line-starting elements that take up the full width of their container.

- Inline tags flow within a line of text and only occupy the space needed by their content.

2. What is the margin, border, and padding?

- Margin = the space outside an element's border.

- Border = the visible line that surrounds the content and padding.

- Padding = the space between an element's content and its border.

3. What are units of measurement you can use in CSS?

  • pixels (px)
  • percentages (%)
  • ems (em)
  • rems (rem)
  • viewport width (vw)
  • viewport height (vh)
  • centimeters (cm)
  • millimeters (mm)
  • inches (in)
  • picas (pc)
  • points (pt)

4. When would you use a figure tag?

- When you want to embed a self-contained piece of content like an image, diagram, chart, code snippet, or other visual element within the document.
