Questions

1. What is normal flow in HTML?

It is the default layout where elements appear in the order they are written (block elements: vertically and inline elements: horizontally).

2. What does float do? Why do you need to clear the float?

The float lets elements to be positioned to the left or right and also lets text or other elements wrap around.

Clear the float will prevents other elements from collapsing when that elements are floated.

3. What do absolute, fixed, and relative do?

  • Absolute - Places an element in an exact location on the page and also placed on top of other elements.
  • Fixed - Keeping elements in place when scrolling.
  • Relative - Positions an element relative to its normal position without affecting surrounding elements.

4. How can you make nice navigation buttons for mobile devices?

Change the navigation to a display block layout in CSS, which makes the buttons to have bigger pressable area.
