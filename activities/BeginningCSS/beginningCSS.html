Question

1. Explain the difference between inline, embedded, and external CSS.

  • Inline CSS is written inside an element's style attribute. Example: <p style="color: red;">Test</p>
  • Embedded CSS is placed within a <style> tag in the HTML <head>. Example: <style> p { color: red; } </style>, then just put h1 { color: red; } in the external CSS.
  • External CSS is stored in a separate .css file and linked using <link>. Example: <link rel="stylesheet" href="styles.css">

2. Explain why you should only use external CSS.

External CSS is much easier to maintain and update, and also easy to read and debug.

3. What is a selector used for?

A selector targets HTML elements to apply styles.
