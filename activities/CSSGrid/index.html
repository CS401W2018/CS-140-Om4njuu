2. What is the main difference between CSS Grid and CSS Flex?

- CSS Grid is designed for two-dimensional layouts, which we can arrange elements in both rows and columns.

- CSS Flex is a one-dimensional layout, only arraging elements in a single direction, which can be either rows or columns at a time.
