Questions

1. What you learned about Flex and CSS.

We can arrange elements within a container in a flexible way. There are different type of flex containers and different properties that we can use.

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

2. Why do you need to have an accessible website? What does that mean?

To make sure that everyone can access and use the information on a website including people with disabilities like visual impairments, hearing difficulties, motor impairments, or cognitive challenges.
