Questions

1. The form tag has an "action" attribute. What does it do?

It specifies the URL where the form data will be sent to be processed when the form is submitted.

2. The form tag has a "method" attribute. What does it do?

It specifies the HTTP method used to send form data to the server.

3. What is the difference between a "POST" and a "GET"?

GET is for retrieve data.

POST is for when you want to submit data or sends data to a server for processing.
