2. Explain the difference between an absolute link and a relative link. Why would you use relative links?

  • An absolute link is a hyperlink that contains the full URL to a webpage, including something like "http://" and domain name.
  • A relative link only specifies the path to a page within the same domain/folder, without the full URL (relying on the current page's location to find the target page).
  • The reason I would use relative link is because I don't need to come back and edit the link if the page is updated in the future (no techically required).

4. List the 3 rules with relative links.

  1. Have to be in the same folder/domain, then use the file name, for example, file.html
  2. If you want to access a file in a subdir, you have to specify the dir path after the filename, for example, folder/file.html
  3. If you want to nav up to diff folder (not sub folder), use ".." then slash, for example, ../previous-page.html or ../folder/previous-page.html
