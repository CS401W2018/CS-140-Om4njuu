Questions

1. Why don't we use the border attribute?

Nowadays it's uncommon to style elements using html border attributes. It's better to apply styling through CSS.

2. What is the easiest way to color every other row a different color?

Use the :nth-child(odd) or :nth-child(even) (pseudo classes) to target rows and apply different background colors.
