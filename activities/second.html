  1. Internet

    2. The internet is a global network that lets users to access information, communicate, and interact with each other.

  2. Web

    3. The web, or World Wide Web (WWW) is a system of web pages and other resources that are accessible over the internet.

  3. How are the Internet and Web different?

    4. The internet is the network that connects millions of computers together. The World Wide Web is the part which you access through your web browser.

  4. ASCII and Unicode

    5. ASCII (American Standard Code for Information Interchange) is a simple way for computers to represent letters, numbers, and symbols using a unique code number for each character.

    Unicode is a universal system that assigns a unique code to every character from all languages in the world.

  5. What is UTF-8

    6. UTF-8 is a way of storing text on computers that lets almost any language in the world, by making common characters like English letters take up less space.

  6. URL

    7. A URL (Uniform Resource Locator) is an address of the webpage.

  7. Domain Name

    8. A domain name is a string of characters that uniquely identifies a resource on the internet.

  8. HTTP

    9. HTTP (Hypertext Transfer Protocol) is the base of the World Wide Web, and is used to load webpages using hypertext links.

  9. HTTPS

    10. HTTPS (Hypertext Transfer Protocol Secure) is a secure way to transfer data on the internet, like when you visit a website. The "S" at the end means "secure."

  10. IP Address

    11. An IP address is a number that uniquely identifies a device on the internet.

  11. DNS

    12. The DNS (Domain Name System) turns domain names into IP addresses, which browsers use to load internet pages.

  12. W3C Recommendations

    13. A W3C Recommendation is a standard or guideline that sets technical specifications and best practices for web technologies.

  13. XML, XHTML, HTML, HTML5

    14. HTML (HyperText Markup Language) is a foundational language for building web pages that let users to create elements like headings, paragraphs, links, images, etc.

    XHTML (Extensible HyperText Markup Language) is a version of HTML that follows XML rules, requiring all elements to be properly closed and nested, making it more structured and machine-readable

    XML (Extensible Markup Language) is a language for defining custom data structures, that let users create their own tags to represent information in a structured way.

    HTML5 (HyperText Markup Language version 5) is the latest version of HTML with new features like video and canvas elements, improved tagging, and better support for web applications.
