Structural Tags

Question 1: Structural Tags (HEADER, NAV, MAIN, FOOTER).

HEADER is the top part of the page.

NAV is the navigation bar that could go in the header.

MAIN is the main content of the page.

FOOTER the bottom of the page.

Question 2: What goes in the header?

Something like a company logo, company name, site title, navigation menu/bar, search bar, and sometimes contact details.

Question 3: What goes in the Nav? Can the Nav go in the header? Does it have to go in the header? Does the Nav contain links to another website?

Nav is lists of links that help users navigate through the website.

Yes, Nav can go in the header.

Nav not nescessary need to go inside the header.

Yes, Nav can contain links, it could be either absolute link or relative link.

Question 4: What goes in Main?

Main content of the webpage should be inside Main

Question 5: What goes in the Footer?

Something like a copyright notice, contact information, and navigation links.

Question 6: HTML Validation

HTML validation is the process of checking HTML code to make sure that it follows the HTML specification which this helps identify errors in the code, ex. missing tags or incorrect attributes.
