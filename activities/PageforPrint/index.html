Questions

1. Why is the viewport meta tag important?

Because it helps the website to displays properly across different devices.

2. How can you adjust the CSS for printing?

Add <link rel="stylesheet" href="print.css" media="print"> in the head section.

3. When would you want to adjust the CSS for printing?

When you want to optimize the layout and appearance of the web page specifically for a printed document.
